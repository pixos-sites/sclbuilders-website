# Image Guidelines

You can use either remote image URLs or local images.

## Preset Local Image Folders

- `/Users/gavinpiggot/Desktop/Master_Template/public/images/plumber/`
- `/Users/gavinpiggot/Desktop/Master_Template/public/images/electrician/`

Reference local images inside presets with:

- `/images/plumber/<filename>`
- `/images/electrician/<filename>`

## Recommended Sizes

- Hero image: 1920x1080
- Service images: 800x600
- Portfolio images: 1080x1080

## Tips

- Prefer JPG/WebP for photos.
- Keep image files optimized for fast loading.
- Use descriptive filenames to match preset content.
