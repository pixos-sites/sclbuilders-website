# Image Guidelines

Add your project photos to this folder. Here's what you'll need:

## Hero Section (Main Banner)
- **Filename**: `hero.jpg` or `hero.png`
- **Recommended size**: 1920x1080px (landscape)
- **Description**: Your best/most impressive project photo

## Portfolio Section (7 project photos)
- **Filenames**: `project1.jpg`, `project2.jpg`, ... `project7.jpg`
- **Recommended size**: 1080x1080px (square works best)
- **Projects**:
  1. House Extension
  2. Kitchen Renovation
  3. Bathroom Renovation
  4. Loft Conversion
  5. Open Plan Living
  6. Garden Landscaping
  7. (Any additional project)

## Services Section (7 service photos)
- **Filenames**: `service1.jpg`, `service2.jpg`, ... `service7.jpg`
- **Recommended size**: 800x600px
- **Services**:
  1. Extensions & Conversions
  2. Kitchens & Bathrooms
  3. New Builds
  4. Roofing & Joinery
  5. Interior Trades
  6. Landscaping & Groundworks
  7. UPVC Windows & Doors

## Tips:
- Use JPG format for photos (smaller file size)
- Optimize images before uploading (keep under 500KB each)
- Use landscape or square orientations
- Make sure photos are well-lit and professional

Once you add the images, let Claude know and they'll update the code to use your photos!
